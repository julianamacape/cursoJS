Manipulação de Dados

Nessa página utilizamos mais uma vez o comando window.prompt e vamos fazer uma análise a respeito dessa soma que foi sugerida. 

        
            <script>
                var n1 = window.prompt ('Digite um número: ')
                var n2 = window.prompt ('Digite outro número: ')
                var soma = n1 + n2
                window.alert ('A soma dos dois valors que você digitou é ' + soma)
            </script>

Veja o código acima. A questão é que o comando window.prompt sempre retorna uma string, mesmo que o que foi digitado tenha sido um número, sabia disso? É por isso que a última janela pop-up não retornou a soma, ela simplesmente colocou juntinho os 2 valores que você digitou, ou seja, foi feita uma concatenação.
É preciso, nesse caso, tratar os dados de n1 e n2. É preciso converter esses dados.
