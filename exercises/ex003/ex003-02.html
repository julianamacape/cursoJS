Manipulação de Dados

Nessa página nós convertemos n1 e n2 para number, por isso que a soma funcionou no comando window.prompt.
Veja o código abaixo 

        
            <script>
                var n1 = Number.parseInt(window.prompt ('Digite um número: '))
                var n2 = Number.parseInt (window.prompt ('Digite outro número: '))
                var soma = n1 + n2
                window.alert ('A soma dos dois valors que você digitou é ' + soma)
            </script>

Ou você pode simplesmente digitar Number(n) e o JavaScript lida bem sem aperreio se for inteiro ou real. Ficaria algo tipo o código abaixo: 

        
        var n1 = Number(window.prompt ('Digite um número: '))
        var n2 = Number(window.prompt ('Digite outro número: '))
        var soma = n1 + n2
        window.alert ('A soma dos dois valors que você digitou é ' + soma)
