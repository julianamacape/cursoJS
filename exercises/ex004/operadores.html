Operadores Aritméticos

  • adição +
  • subtração -
  • multiplicação *
  • divisão /
  • resto da divisão inteira %
  • potência **

Ordem de precedência

  1. tudo o que estiver entre ()
  2. potência
  3. divisão/multiplicação/resto da divisão
  4. soma/subtração

E se houver mais de uma operação dos tipos 3 ou 4 numa mesma expressão, comça a resolver da esquerda para a direita.

Auto-atribuição

Veja que no código abaixo, n inicialmente vale '3', mas aí passa a valer '7' em função da auto-atriuição. 

        
            > var n = 3
            undefined
            > n
            3
            > n = n+4
            7
            > n
            7

Vamos continuar brincando no NodeJS e vendo as auto-atribuições acontecendo: 

        
            > n
            7
            > n = n-5
            2
            > n
            2
            > n = n*4
            8
            > n
            8
            > n = n/2
            4
            > n
            4
            > n = n**2
            16
            > n
            16
            > n = n%5
            1
            > n
            1

Auto-atribuições simplicadas

É uma forma de encurtar a expressão então, por exemplo, ao invés de escrever n = n+4 você escreve n+=4. Veja abaixo no NodeJS as mesmas auto-atribuições que fizemos na demonstração passada, agora simplificadas. 

        
            > n
            7
            > n = n-5 ➡ ficaria n+=5
            2
            > n
            2
            > n = n*4 ➡ ficaria n*=4
            8
            > n
            8
            > n = n/2 ➡ ficaria n/=2
            4
            > n
            4
            > n = n**2 ➡ ficaria n**=2
            16
            > n
            16
            > n = n%5 ➡ ficaria n%=5
            1
            > n
            1

Operadores de Incremento

Quando acontecer de ser n = n+1 ou n = n-1 já aprendemos que, através do comando da auto-atribuição simplificada, a expressão pode ficar n+=1 ou n-=1, respectivamente.
Com os operadores de incremento, você pode transformar essa empressão em x++ ou x--, respectivamente.
