Comando document.write

Usamos o comando document.write para escrever na tela usando JavaScript. É o equivalente ao escreva do Portugol (VisualG) 

        
            <script>
                var nome = window.prompt('Qual o seu nome?')
                document.write(`Seu nome tem ${nome.length} letras.`)
            </script>

Também é possível mesclas tags de html aos comandos de JavaScript. Foi o que fizemos com o <h2> abaixo: 

        
            <script>
                document.write(`
Seu nome tem ${nome.length} letras.
`)
            </script>

Formatação de números

Vamos ver abaixo o código que formata a quantidade de casas decimais nas variáveis tipo NUMBER 

        
            > var n1 = 1545.49582
            undefined
            > n1
            1545.49582
            > n1.toFixed(2)
            '1545.50'

Vamos ver abaixo o código que muda o sinal de casa decimal de ponto para vírgula 

        
            > n1.toFixed(2)
            '1545.50'
            > n1.toFixed(2).replace('.',',')
            '1545,50'

Vamos ver abaixo o código que define e/ou muda a moeda, no caso de valores monetários 

        
            > n1.toLocaleString('pt-BR',{style:'currency',currency:'BRL'})
            'R$1,545.50'
            > n1.toLocaleString('pt-BR',{style:'currency',currency:'USD'})
            '$1,545.50'
            > n1.toLocaleString('pt-BR',{style:'currency',currency:'EUR'})
            '€1,545.50'
