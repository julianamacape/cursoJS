Aprendendo JavaScript

Termos relacionados a variáveis em JavaScript

= ➡ recebe
null ➡ nulo (vazio)
var ou let ➡ variável
string ➡ é texto. O string tem que vir dentro de aspas. Há 3 possibilidades:

  1. "Texto"
  2. 'Texto'
  3. `Texto`

Identificadores de variáveis

  • podem começar com letra, $ ou _
  • não pode começar com números
  • pode mesclar letras e números
  • pode usar acentos gráficos e símbolos
  • não pode usar espaços
  • não pode ser uma palavra reservada
