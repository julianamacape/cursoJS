Aprendendo JavaScript

Termos relacionados a variáveis em JavaScript

  • = ➡ recebe
  • null ➡ nulo (vazio)
  • var ou let ➡ variável
  • string ➡ é texto. O string tem que vir dentro de aspas. Há 3 possibilidades:
  1. "Texto"
  2. 'Texto'
  3. `Texto`

Identificadores de variáveis

  • podem começar com letra, $ ou _
  • não pode começar com números
  • pode mesclar letras e números
  • pode usar acentos gráficos e símbolos
  • não pode usar espaços
  • não pode ser uma palavra reservada

Pra entrar no NodeJS no MacOS, há 2 maneiras:

  1. abra o Terminal, digite node - v
  2. No VS Code ➡ Terminal ➡ New Terminal ➡ digite node - v

Para sair do NodeJS

  • digite .exit
  • para sair do terminal, digite exit

⚠ Não abra vários terminais dentro do VS Code, pois isso vai deixar o sistema super lento ⚠

⚠ Lá no NodeJS, pra você saber qual é o tipo da variável já declarada e identificada (ou um valor qualquer sem que seja variável), usamos o comando dentro do NodeJS chamado: typeof ⚠ 

        
            > var n = 200
            undefined
            > n
            '200' 

            > var n = 'Juliana'
            undefined
            > n
            'Juliana'
            > typeof n
            'string' 

            > typeof 6
            'number'
            > typeof '6'
            'string'
            > typeof []
            'object'
            > typeof {}
            'object'
            > typeof function (){}
            'function'
